# Calculator App

A simple and responsive calculator built with **HTML**, **CSS**, and **JavaScript**. It performs basic arithmetic operations with a clean and modern user interface.

## Features

- Addition, subtraction, multiplication, and division
- Decimal number support
- Clear (`C`) button
- Simple and responsive design
- Instant calculation

## Technologies Used

- HTML5
- CSS3
- JavaScript
